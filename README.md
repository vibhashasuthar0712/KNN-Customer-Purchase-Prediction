# 🚀 KNN Customer Purchase Prediction

## 📌 Overview

This project uses the **K-Nearest Neighbors (KNN)** algorithm to predict whether a customer will purchase a product based on:

* Age
* Salary



## 🧠 Problem Statement

Given customer data, predict:
👉 Will the customer purchase the product or not?



## ⚙️ Tech Stack

* Python 🐍
* Pandas
* NumPy
* Scikit-learn
* Matplotlib


## 🔍 Workflow

1. Data Loading
2. Data Preprocessing
3. Feature Selection
4. Train-Test Split
5. Feature Scaling
6. Model Training (KNN)
7. Model Evaluation
8. Visualization (Decision Boundary)
9. Prediction on New Data



## 📊 Model Details

* Algorithm: K-Nearest Neighbors (KNN)
* K Value: 3
* Distance Metric: Euclidean
